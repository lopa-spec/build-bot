# Build Bot - Flutter APK Builder 🚀

Bot Telegram untuk build APK Flutter otomatis via GitHub Actions.

## 📌 Fitur
- Build APK dari ZIP project Flutter
- Support Debug & Release mode
- Web to APK converter
- Multi-user dengan priority system

## 🔧 Cara Pakai
1. Kirim /start ke bot
2. Pilih mode build
3. Kirim file ZIP project Flutter
4. Tunggu APK jadi!
